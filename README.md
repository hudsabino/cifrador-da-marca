# Cifrador da Marca

Aplicativo web que converte palavras no cifrario do Grau de Mestre Macom da Marca,
segundo a chave de grade espelhada. Obra original de **Hudson Sabino**, identificada
pela marca pessoal **HCSLA**.

## Conteudo da pasta

- `index.html` ................ o aplicativo inteiro, em arquivo unico
- `manifest.webmanifest` ...... permite instalar como aplicativo no celular
- `sw.js` ..................... faz o aplicativo funcionar sem internet
- `icone-*.png`, `favicon.svg`, `apple-touch-icon.png` ... icones, gerados a partir da marca HCSLA
- `LICENSE.txt` ............... declaracao de autoria e condicoes de uso

## Instalar no celular

Abra o endereco no Safari (iPhone) ou no Chrome (Android) e escolha
"Adicionar a Tela de Inicio". O aplicativo passa a abrir em tela cheia e funciona
offline.
